YT Summarizer Pro

Full-stack AI app that summarizes any YouTube video and enables natural-language Q&A over its transcript. Built a 4-layer fault-tolerant transcript pipeline (youtube-transcript-api → yt-dlp → AssemblyAI → Supadata) supporting Telugu, Hindi, and English. Transcripts are embedded into a FAISS IndexFlatIP vector store via SentenceTransformers (all-MiniLM-L6-v2) for semantic retrieval, with Groq LLaMA-3.3-70b powering instant summaries and context-grounded Q&A — achieving sub-3-second end-to-end execution. Dockerized with GitHub Actions CI/CD.
